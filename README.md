# Chess

A real-time two-player chess game using Express, Socket.IO, EJS, and chess.js.

## Run

```bash
npm install
npm start
```

Open `http://localhost:3000` in two browser windows. The first player gets White, the second gets Black, and extra visitors can watch as spectators.

## Features

- Server-validated legal moves
- White, Black, and spectator roles
- Responsive board UI
- Mobile-friendly tap-to-move controls
- Pawn promotion picker for queen, rook, bishop, or knight
- Check, checkmate, stalemate, and draw status
- New game and board flip controls
